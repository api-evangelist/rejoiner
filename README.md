# RejoinerThis is a repo for managing machine readable artifacts for the [Rejoiner APIs](http://rejoiner.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Rejoiner APIs](http://rejoiner.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://rejoiner.com)- [Developer](http://rejoiner.com)- [Twitter](https://twitter.com/rejoinerapp)- [GitHub](https://github.com/rejoiner)The [APIs.json](https://github.com/api-evangelist/rejoiner/blob/master/apis.json) index provides links to all the relevant links for the Rejoiner APIs, including the OpenAPI and Postman if they are present.
